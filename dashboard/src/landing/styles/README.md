# CSS Styles in Landing Page

This folder contains the styles for the landing page.

- `landing.css`: Contains keyframe animations, scrollbars, global overrides, and standard responsive utilities.
- Section components should use inline CSS or CSS-in-JS where possible to keep them modular, encapsulated, and easier to scale.
- Reusable class names are prefixed with `evy-` or `calc-` for marquee track effects.
